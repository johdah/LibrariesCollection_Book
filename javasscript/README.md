## Visualization

### Epoch
A general purpose real-time charting library for building beautiful, smooth, and high performance visualizations.

```
http://fastly.github.io/epoch/
```

Supported graphs:
* Area
* Bar
* Line
* Pie
* Real-time Area
* Real-time Bar
* Real-time Heatmap
* Real-time Line
* Scatter
